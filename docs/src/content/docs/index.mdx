## Features

<CardGrid stagger>
  <Card title="Per-instance frustum culling">Skip rendering for out-of-view instances.</Card>
  <Card title="Sorting">Reduce overdraw and manage transparent objects efficiently.</Card>
  <Card title="Spatial indexing (dynamic BVH)">Speed up raycasting and frustum culling.</Card>
  <Card title="Dynamic capacity">Add or remove instances seamlessly.</Card>
  <Card title="Per-instance visibility">Toggle visibility for each instance individually.</Card>
  <Card title="Per-instance opacity">Set opacity for each instance individually.</Card>
  <Card title="Object3D-like instances">Use instances like Object3D with transforms and custom data.</Card>
  <Card title="Per-instance uniforms">Assign unique shader data to individual instances.</Card>
  <Card title="Level of Detail (LOD)">Dynamically adjust instance detail based on distance.</Card>
  <Card title="Shadow LOD">Optimize shadow rendering with lower detail for distant instances.</Card>
</CardGrid>
